###  Instalação


`npm install`

####  Docker

Existe um arquivo `docker-compose.yml` para iniciar o Docker com o seguinte comando:

`docker-compose up`

Depois de executar a amostra, você pode parar o contêiner do Docker com:

`docker-compose down`

###  Execute a amostra

Em seguida, execute o Nest normalmente:

`npm run start`
